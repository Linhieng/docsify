# docsify 提供的一些 markdown 语法

## 红色强调

```md
!> 一段重要的内容，可以和其他 **Markdown** 语法混用。
```

!> 一段重要的内容，可以和其他 **Markdown** 语法混用。

## 普通提示

```md
?> _TODO_ 完善示例
```
